About DoBackChess

DoBackChess is free chess software developed by Eastern Kentucky Digital ↗. It is currently a public alpha, so features may change while we continue testing engine behavior, reports, browser support, and account sync.

Stockfish and the Reckless browser alpha run locally on your computer. Signed-in Google and email/password accounts can retain a larger Firebase-backed game library and sync reports, preferences, and review progress.

Planned features

  • Lc0 analysisNeural-network analysis as an additional perspective.

  • Endgame studyPractice important technical endings from real games.
  • Deeper history toolsUse larger long-term game libraries to identify durable patterns.

Analysis engine

Analysis strength

Stronger analysis is slower. Higher settings can take several times longer and use more CPU and battery. You can stop training or report analysis at any time; stopped training keeps puzzles already completed.

Games per report

20 games per Chess Report or Tactics Report. Training still includes every game from the last 7 days when that is larger.

More games take proportionally longer. A 50-game report can take about two and a half times as long as the 20-game default.

All beta analysis features run locally and are included. Reckless browser alpha may download about 61.5 MiB when first started. No chess positions leave this device. Browser source · engine source · AGPL-3.0 license.
